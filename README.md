# Patrick O'Brien's Blog Site

## Prerequisites:

- yarn
- NodeJS >= 13.5.0

## Build

```shell
> yarn
> yarn run build
```

## Local development

```shell
> yarn
> yarn run build # need to generate some additional files (e.g. parcel output, service worker, etc.)
> yarn run serve
```
